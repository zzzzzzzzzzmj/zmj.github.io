4 个鲜为人知的 GitHub 神技巧

发表于 2022-05-25 分类于 GitHub 阅读次数：
本文字数： 402 阅读时长 ≈ 1 分钟
本文基于 B 站 UP 主 @程序员鱼皮 的《 5 个隐藏的 GitHub 神技巧，助你秒变大佬！》视频总结而成，非常厉害，非常实用。

⭐️ 在线阅读代码

在仓库详情界面，按下 . 键，即可利用在线的 VSCode 阅读代码，而无需下载到本地。

⭐️ 在线运行项目

在项目的浏览器地址前，加上 gitpod.io/#/ 前缀，即可在线运行项目。

实用快捷键总结

查找文件

在仓库详情界面，按 T 键，然后输入要查找的文件名。

跳转至某行代码

**在源代码界面，按 L 键，然后输入要跳转至的行号。**还可以：

  • 复制该行代码

  • 生成永久链接（permalink）

查看文件改动记录

在源代码界面，按 B 键，即可查看该文件的改动记录。

更多

除此之外，GitHub还有超多快捷键，详见 官方文档。

订阅 GitHub 项目推送

在 GitHub 中的 Explore 页面中，点击 Get email updates，即可让 GitHub 定期地推送自己可能感兴趣的优质项目，到自己的邮箱，以了解开源动态。
